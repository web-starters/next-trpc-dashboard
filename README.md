# next-trpc-dashboard

Next.js dashboard with tRPC

## Configuration

You have to create an `.env` file in the root directory like `.env` file.

## Development

Install dependencies:

```shell script
pnpm install
```

Start development mode:

```shell script
pnpm run dev
```

Build project:

```shell script
pnpm run build
```

Start production:

```shell script
pnpm run start
```
